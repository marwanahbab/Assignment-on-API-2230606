About Country Information Database

Country Information Database is a web application designed to help users discover information about countries around the world. Using data from the REST Countries API and weather information from Open-meteo,

Data Sources

This application utilizes the following APIs to provide accurate and up-to-date information:

  • REST Countries API - Provides comprehensive country data including flags, population, and geographical information.
  • Open-meteo - Supplies current weather conditions for each country.

Features

  • Search for countries by name
  • View basic information about countries in a card layout
  • Access detailed country information including:
    • Official name and native name
    • Population statistics
    • Geographic details (area, region, subregion)
    • Currency information
    • Official languages
    • Current weather conditions in the capital city
  • Interactive map showing country location
  • Responsive design that works on mobile devices and desktops

Developed as a project for learning web development and API integration.
